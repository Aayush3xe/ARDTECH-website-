## Step 1: Set Up Free Hosting

3. Enter your email: **support.ardtech@gmail.com**

1. Log in to InfinityFree
2. Click "Create Account" 
3. Select subdomain (e.g., **ardtech.42web.io**)
4. Click "Create"

## Step 2: Upload Your Website

### Using FileZilla (Free FTP Client):
1. Download FileZilla: https://filezilla-project.org/
2. Open FileZilla
3. Go to Site Manager → New Site
4. Settings:
   - Host: ftpupload.net
   - Protocol: FTP
   - Username: (check InfinityFree email)
   - Password: (your password)
5. Connect
6. Navigate to **htdocs** folder
7. Upload ALL files from this folder

## Step 3: Set Up WhatsApp Cloud API (FREE)

### Create Meta Developer Account:
1. Go to: https://developers.facebook.com/
2. Click "Create App"
3. Select **Business** as app type
4. Name: **ARD Tech**
5. Click "Create App"

### Add WhatsApp Product:
1. In your app dashboard, click "Add Products"
2. Find "WhatsApp" → Click "Set up"
3. Select or create a Business Manager account
4. Follow the prompts

### Get Your Credentials:
1. Go to **WhatsApp** → **API Setup**
2. Copy these values:
   - **Phone Number ID**
   - **Temporary Access Token**

### Configure the Webhook:
1. Open **whatsapp-webhook.php** in a text editor
2. Replace these values:
```php
$CONFIG = [
    'PHONE_NUMBER_ID' => 'YOUR_PHONE_NUMBER_ID',  // e.g., '123456789'
    'ACCESS_TOKEN' => 'YOUR_ACCESS_TOKEN',          // Long string from Meta
    'VERIFY_TOKEN' => 'ardtech2024securetoken',    // Keep this
    'ADMIN_PHONE' => '27653547585',               // Your WhatsApp
    'BUSINESS_NAME' => 'ARD Tech'
];
```

3. Save the file
4. Upload the updated file to your hosting

### Configure Webhook in Meta:
1. In WhatsApp API Setup, click "Edit"
2. Enter:
   - **Callback URL**: https://yourdomain.com/whatsapp-webhook.php
   - **Verify Token**: ardtech2024securetoken
3. Click "Verify and Save"
4. Under "Webhooks", click "Manage"
5. Subscribe to: **messages**
